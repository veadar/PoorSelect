#PoorSelect

##About

PoorSelect find items which match with specified condition at selected location in Finder , and select them.(This is inspired by PowerSelect)

##Usage

- Add Finder Toolbar
- Use from LaunchBar or Alfred.

![Anime Gif](./PoorSelect.gif)

Example

- ^[0-9]
- png$
- -d txt
- -i hoge
- hoge | grep fuga
- e hoge -e fuga
